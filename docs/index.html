This page lets you visualize drop voicings (close, drop 2, drop 3, drop 2+3, drop 2+4) for any set of intervals.
Select intervals, pick a voicing, choose a valid string set and the tool will render the chord shape plus its inversions.

Choose up to 6 unique intervals (unordered). Order and voicing are applied later.
